# Brain_inference 

An fMRI data analysis project focusing on brain inference and stimulus dimension analysis. 

## Overview
This repository contains scripts for processing and analyzing functional Magnetic Resonance Imaging (fMRI) data. The primary focus is to investigate brain stimulation dimensions.

##  Prerequisites
To use the scripts in this repository, you will need:
* **MATLAB** (Tested on version R2025b)
* **SPM12** (Statistical Parametric Mapping) toolbox installed and added to your MATLAB path.
